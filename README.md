# PropMaster
A .prop creation tool for SpyParty event organizers and helpers.  These .prop files make a quickplay group in the SpyParty game, allowing easy switching between venue setups within a particular rule set.

PropMaster allows simple configuration of most features of .prop files, and files created using the tool also have some additional useful documentation embedded in comments at the end of the file.  

For use of PropMaster, [it is currently hosted on bl00dw0lf.com](https://bl00dw0lf.com/SpyParty/PropMaster/)
