SpyParty .prop making tool for event organizers.


Notes: Test your .prop file before you release it! This tool does not account for all possible errors. Prop files should be installed in the %LOCALAPPDATA%\SpyParty\quickplays\ folder, which you may need to create.
Hints: For Pick or Any gamemodes, make sure the # of missions needed to win is less than the # of selected missions, or the prop file will not load. Ensure each venue has at least the minimum amount of guests required for all missions (4-6 depending on enabled missions. This isn't error checked yet.).
Pre-filled data is primarily based on SCL6 (without default selections like on balcony).
